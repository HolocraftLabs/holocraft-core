# Holocraft Style Taxonomy

Holocraft maintains a structured library of visual languages.

Each style direction defines a complete design identity capable of generating brand-specific digital experiences.

The Style Taxonomy exists to:

- prevent visually identical product outcomes
- provide AI-readable design direction
- create reusable creative frameworks
- guide research extraction and component generation

---

# How to Use This Taxonomy

1. Define the client's emotional direction and brand personality
2. Select a primary style from this taxonomy
3. Reference the corresponding research documents for that direction
4. Extract typography, motion, spacing, and color systems from research
5. Implement using design tokens and motion primitives

A project may combine at most two styles.

Document the selected direction in the project architecture file before any implementation begins.

---

# Cinematic

Emotional direction: immersive, dramatic, high-production-value, storytelling-driven

Visual characteristics:
- dark dominant palette
- full-bleed imagery and video
- bold typographic scale contrast
- layered compositional depth
- intentional use of light and shadow

Motion characteristics:
- slow deliberate entrance reveals
- parallax depth layers
- cinematic section transitions
- ambient floating elements
- scroll-driven narrative progression

Typography behavior:
- oversized display headlines
- high contrast weight variation
- editorial column widths
- expressive display typefaces

Reference industries: film, music, luxury entertainment, high-end creative agencies

---

# Editorial

Emotional direction: sophisticated, intellectual, culture-forward, story-driven

Visual characteristics:
- high typographic intelligence
- structured grid systems
- controlled whitespace
- monochromatic or limited palette
- photography-led layout composition

Motion characteristics:
- subtle text reveals
- smooth scroll behavior
- content-aware transitions
- restrained, purposeful animation

Typography behavior:
- strong typographic hierarchy
- mix of serif and sans-serif
- editorial column proportions
- wide tracking on display headlines

Reference industries: media, publishing, culture, architecture

---

# Luxury

Emotional direction: premium, exclusive, refined, aspirational

Visual characteristics:
- minimalist layout density
- gold, black, ivory, and deep tones
- generous whitespace
- refined typography
- high-quality photography treatment
- tactile material references

Motion characteristics:
- slow elegant transitions
- subtle fade and reveal
- cursor-reactive interactions
- weighted micro-interactions

Typography behavior:
- high-end serif for display
- light-weight sans for body
- wide letter tracking
- restrained scale variation

Reference industries: fashion, jewelry, hospitality, private banking, luxury automotive

---

# Futuristic

Emotional direction: forward-looking, technological, speculative, optimistic

Visual characteristics:
- sharp geometric forms
- neon accent colors on dark backgrounds
- data-inspired layouts
- grid and line overlays
- technological material references

Motion characteristics:
- glitch and scan effects
- fast sharp transitions
- data-driven animations
- particle and field systems

Typography behavior:
- geometric sans-serif
- monospace for accents and data
- tight letter spacing
- numerical and system-style emphasis

Reference industries: AI, deep tech, aerospace, crypto, developer tools

---

# Brutalist

Emotional direction: bold, confrontational, raw, honest, anti-establishment

Visual characteristics:
- raw grid exposure
- typography as dominant design element
- high contrast black and white with stark accent colors
- visible structure and scaffolding
- anti-polish aesthetic

Motion characteristics:
- abrupt transitions
- scroll-snap behavior
- heavy interactive hover responses
- typographic animation and disruption

Typography behavior:
- oversized display type
- condensed grotesque
- extreme weight contrast
- irregular layout positioning

Reference industries: creative agencies, art direction, music, independent publishing

---

# Minimal

Emotional direction: calm, clarity-focused, purposeful, trustworthy

Visual characteristics:
- maximum whitespace
- monochromatic or near-monochromatic
- single focal point per view
- restrained element density

Motion characteristics:
- subtle opacity transitions
- calm scroll behavior
- no distracting effects
- state-change feedback only

Typography behavior:
- single font family
- clean weight hierarchy
- generous leading
- restrained scale

Reference industries: SaaS, productivity tools, health, wellness, fintech

---

# Cyberpunk

Emotional direction: dystopian, rebellious, neon-drenched, high-energy

Visual characteristics:
- neon color on black backgrounds
- glitch and noise aesthetics
- texture and rain overlays
- fragmented grid layouts
- urban and industrial material references

Motion characteristics:
- glitch loops and chromatic aberration
- fast aggressive transitions
- scan-line reveals
- noise and static overlays

Typography behavior:
- condensed high-tech fonts
- mixed case treatment
- monospace accents
- neon glow effects

Reference industries: gaming, entertainment, underground culture, NFT

---

# Immersive

Emotional direction: enveloping, experiential, surprising, spatial

Visual characteristics:
- full-screen environment sections
- 3D or depth-simulated environments
- background narrative
- environmental storytelling and world-building

Motion characteristics:
- scroll-driven world building
- 3D perspective transforms
- ambient environment motion
- cursor-driven parallax

Typography behavior:
- text as spatial element
- scale variation based on depth
- 3D text treatments where appropriate

Reference industries: gaming, automotive, experiential branding, interactive installations

---

# Motion-Heavy

Emotional direction: kinetic, energetic, dynamic, attention-commanding

Visual characteristics:
- motion as the primary design element
- animated typography
- transition-led navigation
- kinetic layout shifts

Motion characteristics:
- text animation dominance
- stagger entrance systems
- page transition choreography
- persistent ambient motion
- scroll velocity response

Typography behavior:
- animated display type
- morphing letterforms
- rhythm-based reveal patterns

Reference industries: creative studios, entertainment, sports, events

---

# Product-Focused

Emotional direction: functional, trustworthy, efficient, clear

Visual characteristics:
- product visualization as priority
- clean neutral backgrounds
- clear feature hierarchy
- comparison and demonstration layouts

Motion characteristics:
- interface animation
- feature reveal transitions
- product rotation and demonstration
- interface interaction simulations

Typography behavior:
- clear hierarchy at all scales
- readable body text
- marketing headline scale
- feature callout labels

Reference industries: SaaS, consumer products, mobile apps, developer tools

---

# Taxonomy Rules

When selecting a style direction:

- choose based on client brand personality and audience, not personal preference
- a project may blend a maximum of 2 styles
- the primary style must be dominant
- document the selected direction in the project architecture file
- all design decisions must reference the selected direction

The Style Taxonomy is a living document.

New styles are added when the studio identifies new visual patterns through the research system.
