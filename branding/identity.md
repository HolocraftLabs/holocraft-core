# Holocraft Brand Identity

This directory contains brand identity documentation and assets for Holocraft studio itself.

---

# Studio Identity

Holocraft is positioned as:

- an AI-native creative engineering studio
- a design systems laboratory
- a motion-first frontend engineering team
- a cinematic product studio

Not a web agency.
Not a template factory.
Not a traditional software house.

---

# Brand Personality

Holocraft communicates through:

- technological sophistication
- creative intelligence
- premium quality
- forward-thinking perspective
- systems thinking

The brand feels: precise, cinematic, intelligent, future-forward.

The brand does not feel: trendy, generic, playful, or corporate.

---

# Visual Direction

Style taxonomy reference: Cinematic + Futuristic

Color direction: dark dominant, high contrast, selective accent

Typography direction: geometric precision with editorial hierarchy

Motion direction: deliberate, weighted, intentional

---

# Planned Contents

Brand assets to be defined and stored in this directory:

- Logo system (primary mark, wordmark, icon-only)
- Color palette (brand primaries, accents, neutrals)
- Typography system (brand typefaces and scale)
- Motion identity (brand animation style and signature)
- Usage rules and misuse examples

# Status

Brand identity is in development.

Visual assets will be added as the studio brand is formalized.
